Le Nouvel Observateur

Projekt iz kolegija Programiranje web aplikacija.

Autor: Petra Papaik

Pokretanje:
1. Kopirati mapu projekta Le_Nouvel_Observateur u mapu htdocs unutar XAMPP-a.
2. Pokrenuti Apache i MySQL u XAMPP Control Panelu.
3. Otvoriti phpMyAdmin.
4. Kreirati novu bazu podataka pod nazivom:
    le_nouvel_observateur
5. Odabrati kreiranu bazu te kliknuti na Import.
6. Uvesti SQL datoteku:
    database/le_nouvel_observateur.sql
7. Nakon uspješnog importa otvoriti preglednik i pokrenuti projekt na adresi:
    http://localhost/Le_Nouvel_Observateur/
